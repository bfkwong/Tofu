Some Basics on Coding

Types of Values

There are three types of values in programming:

  • Integers are numbers that don't have a decimal place (e.g. 3)
  • Floats are numbers with a decimal place (e.g. 3.14159)
  • Strings are any characters inside quotations (e.g. "pie is 3.14159")
  • Booleans are true or false values (e.g. true)
  • Lists are a collection of different values (e.g. [1, 2.2, "3", "4", false])

Comparing Values

You can compare values with the following operators:

  • == returns a boolean (true or false) value of whether or not two values are equal
  • != returns a boolean (true or false) value of whether or not two values are different
  • > returns a boolean (true or false) value of whether or not the 1st value is greater than the 2nd
  • < returns a boolean (true or false) value of whether or not the 1st value is less than the 2nd
  • >= and <= is pretty much self explanatory

EXAMPLE 

print(3 == 4); 
Output: false 

print(3 == 3); 
Output: true 

print(3 == "3"); 
Output: false

print(3 > 4); 
Output: false

Variable Assignment

You can create variables and assign it a value of any type that we specified above. The way you do it is as follows: 

x = 3;

To the right, we created a variable named x and gave it a value of 3. We used a semicolon to signify that we are done with the line.

We can create a variable of any name and any type. It is important to note that an assignment is a single equal sign but comparing whether two values are the same is a double equal sign.

Once you assign a variable a specific value, you can use that variable as if it was the value. Consider the following example 

x = 3; 
print(x == 3); 
Output: true

Here, we set the variable x to the value of 3 and then we used the comparison operation == to see whether or not x is equal to 3. Since we set x to 3, it is the same, so the statement x == 3 is true.

You can reset the value of a variable the same way you create a variable. Consider the following code. 

x = 3; 
print(3); 
Output: 3

x = 4; 
print(x); 
Output: 4

3. FOR loops

The for loop is a way to enumerate every item in a list. This is the loop you would want to use if you want to do something to a list one by one. Consider the following list of drinks that are on sale at your local grocery market: 

availableDrinks = [
	"Stella Rosa Black", 
	"Smirnoff Ice",
	"White Claw", 
	"3 Buck Chuck", 
	"Jack Daniels", 
	"Trivento Amado Sur"
]

Say we want to write a program that checks whether your favorite drink is available, if it is, you want to print out a message that says "the drink is available". If we were to do this logically, we would go through the list one by one and see if it is our favorite drink. Your thinking might go something like this:

My favorite drink is White Claw, Is the first drink White Claw? No, it is Stella Rosa Black. so we keep going. Is the second drink White Claw? No, it is Smirnoff Ice. so we keep going. Is the third drink White Claw? Yes! The drink is available!

Notice that for every item we come across in the list, we do the same thing: we ask if it is white claw, if it is, then we are done, if it is not, we keep searching.
